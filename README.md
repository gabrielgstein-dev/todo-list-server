## Installation

```bash
docker-compose -f ./docker-compose-dev.yml up -d
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


## Migrations

- Create:
>  npm run migration:create src/migrations/user

- Run:
> npm run migration:run
